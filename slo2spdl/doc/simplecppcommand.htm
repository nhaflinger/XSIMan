SimpleCommand

The simple command plug-in implements a custom command in C++.

Name: C++ Simple Command
Scipting name: SimpleCPPCommand

XSI installs automatically this custom command at start up and also installs a help menu item under Help | C++ Simple Command Help in the main menu bar for viewing this page.

How to Compile and install the C++ simple command plug-in

Windows

Compiling

Open the project in MSDEV, select a compilation target (debug or release) and compile.

Installing

The plug-in is self-installing and doesn't need to be installed explicitly. You only need to set the XSI_PLUGINS environment variable with your plug-in path and XSI will take care of installing it automatically at start up.

The XSI_PLUGINS variable can be set in <XSI installation folder>\Application\bin\setenv.bat as follows:

    set XSI_PLUGINS=%XSI_PLUGINS%;%XSISDK_ROOT%\examplesCommands\SimpleCommand

Linux

Before running any command, you have to set up your XSI environment. To achieve this, run the following command in a shell (tcsh) :

    source [Install dir]/.xsi_[version]

Compiling

In the shell (tcsh),go to the <XSI installation folder>/XSISDK/examples/Commands/SimpleCommand/cpp folder and issue the following command:

    gmake

Cleaning

In the shell (tcsh), issue the following command:

    gmake clean

Installing

No explicit installation is necessary. The variable XSI_PLUGINS takes care of installation. Just remove the comment '#' that block the definition of XSI_PLUGINS for this example in .xsi_version file.
